# v2rayWinUI

A GUI WinUI client for Windows, support [Xray](https://github.com/XTLS/Xray-core)
and [sing-box](https://github.com/SagerNet/sing-box)
and [others](https://github.com/2dust/v2rayN/wiki/List-of-supported-cores)

## How to build

- Clone the repository and open V2RayN**.slnx** file in Visual Studio 2026.
- Restore NuGet packages.
- Build the solution with no pacckaged.
- You may need to put xray-core or sing-box binary in the out-put bin directory.
